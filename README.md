# Messier Object Viewer

A collection of tools to explore the Messier catalog. Includes a **web app** (HTML/CSS/JS) and a **Python desktop app** (Tkinter). The **Python desktop app** is not up-to-date. 

## Features
- Filter by constellation
- Sort by magnitude, difficulty, or Messier number
- View object images

## Access
- **Web app:** https://angelos-th.github.io/Messier-Objects/
